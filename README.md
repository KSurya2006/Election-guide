# 🗳️ Civic Learner: Election Process Guide

Welcome to the **Civic Learner**, a premium, single-page interactive web application designed to educate first-time voters and citizens about the election process. 

## ✨ Features
- **Dynamic Onboarding:** Personalize your learning experience based on your background and country (USA, India, UK, or General Demo).
- **My Civic Score:** A gamified tracker that increases as you interact with the app. Reach 100 points to earn the "Civic Champion" badge and a confetti celebration!
- **Interactive Timeline:** A visually engaging, scrollable timeline with pulsing animations that explains the step-by-step electoral process.
- **Learn Cards:** Expandable lessons containing helpful analogies, key dates, and actionable checklists.
- **Knowledge Quiz:** A 10-question, country-specific quiz with a built-in 15-second timer per question and a shareable result card.
- **Election Glossary:** An A-Z filterable glossary with 3D flip-cards to demystify complex political terms.

## 🎨 Design & Tech Stack
- **Tech Stack:** 100% Vanilla HTML, CSS, and JavaScript. No external frameworks required.
- **Design Language:** Built using **Google Material Design 3** principles, featuring:
  - Responsive layout
  - Elegant elevation shadows and rounded cards
  - Authentic "ripple" click animations
  - Built-in Dark Mode toggle
  - Vibrant "Google" color palette (Blue, Yellow, Green, Red)

## 🚀 How to Run
Since this is a fully self-contained single-page application, no installation or server is required! 
1. Download or clone this repository.
2. Double click the `election-guide.html` file to open it in any modern web browser.
3. Start learning!
